## Getting Started

First, run the development server:

```bash
npm run develop
# or
yarn develop
```

For build:

```bash
npm run build
# or
yarn build
```

To start build on the server:

```bash
npm run start
# or
yarn start
```
Open [http://localhost:1337/admin](http://localhost:1337/admin) with your browser to see the result.
